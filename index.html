Game Description:

This project is a chess game and requires 2 users. Both players enter in their name and then a game is begun. The players take turns moving until they are finished.

The Model Description

  • The user clicks the Play Button
  • The game object asks each player for a name. Names will displayed to show who's turn it is.
  • A loop (most likely foreach) will place all the pieces in their appropriate location to start.
  • When a piece is selected the appropriate moves are displayed.
  • When an optional move is clicked the piece object remembers its new location
  • The piece object then informs the game object
  • The game object will tell the board object and the pieces to switch sides allowing the same view for both players. Each player object will run through a loop (most likely foreach) to propagate the switch.
  • When a piece is taken it is marked as taken and is removed from the array of piece objects held by the player object. Can use CSS or actually delete object from the array.
  • Subsequent moves continue until the finished button is selected.
  • The game object then asks if the users want to play another game or quit.
  • The game object will reset the game as indicated or simply leave the page with the completed game in tact.
