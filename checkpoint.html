Completed Game Logic

  • Function that switches turns - nextTurn()

  • Functinon that checks to see what is in a square (what color or if the square is empty) - checkSquare()

  • Function that gets the color of the piece - getColor()

  • Function that highlights possible moves - highlight()

  • Function that moves whatever piece to a new location - Move()

  • Functions that remove event listeners and highlights removeHighlight()

Instructions To Try Out The Game

To test these functions simply click on a pawn.
The piece and its possible moves will be highlighted.
After the first piece is moved then you will be able to click on a black pawn and move it.
The game cannot yet enforce who's turn it is, nor are any pieces ready to be moved except for pawns.
But basically highlight and moving functionality has been built.
Many of the functions to do this will be resused when it comes to moving other pieces as well.
Its important to note that the pawns can't move diagonally yet.
